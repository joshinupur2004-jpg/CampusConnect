# Campus Connect

A full-stack student networking platform built with Flask, MySQL, and Socket.IO.

## Features
- Student Registration & Authentication
- Profile Management (Bio, Skills, Image Upload)
- Connection Request System (Send/Accept/Remove)
- Real-time Chat (Socket.IO)
- Student Search & Discovery
- Dashboard with Analytics

## Setup Instructions (Local)

### 1. Prerequisites
- Python 3.9+
- MySQL (XAMPP or Local Installation)
